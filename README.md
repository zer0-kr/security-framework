# security-framework

글로벌 정보보호 프레임워크(NIST, OWASP 등)의 문서 체계를 정리하고, 각 문서의 핵심 내용을 한국어로 요약하여 제공하는 프로젝트입니다.

## 구조

```
security-framework/
├── NIST/       ← NIST 사이버보안 및 프라이버시 문서
└── OWASP/      ← OWASP 보안 가이드 (예정)
```

## NIST

NIST(미국 국립표준기술연구소)의 사이버보안/프라이버시 출판물을 다룹니다.

- [NIST 문서 체계 가이드](./NIST/README.md) — NIST 문서 구조, 시리즈별 설명, 핵심 프레임워크(CSF 2.0, SP 800-53 등) 해설
